# gamp — 미니게임 천국 (웹 버전) 기본 소스

### 팀원: 박상민, 최원욱, 한세윤

간단한 캔버스 게임 프레임워크와 3가지 미니게임(러너, 캐쳐, 클리커) 샘플이 포함된 스타터 프로젝트입니다. 메뉴에서 하나를 선택해 바로 플레이할 수 있습니다.

## 구성
- `index.html` — 캔버스와 메뉴/HUD 오버레이 UI, 스크립트 로딩
- `style.css` — 기본 스타일
- `js/engine.js` — 캔버스 초기화, 입력, 충돌 등 유틸 + 씬 매니저
- `js/games.js` — 3가지 미니게임 구현 (Runner, Catcher, Clicker)
- `js/main.js` — 메뉴/게임 씬 연결, HUD 및 로컬 최고점 저장

## 조작
- 공통: R 키 — 메뉴로 돌아가기
- 러너: 스페이스 / ↑ / 터치 — 점프
- 캐쳐: ← → / 마우스 드래그 — 좌우 이동
- 클리커: 마우스/터치 — 파란 목표 클릭, 제한시간 30초

## 실행 (Windows, 로컬)
아래 중 편한 방법을 선택하세요.

### 1) 바로 열기 (가장 쉬움)
- `index.html` 파일을 더블클릭해 브라우저로 엽니다.
  - 모듈/네트워크 요청이 없도록 설계되어, 서버 없이도 작동합니다.

### 2) 간단 서버로 열기 (선택)
파이썬이 있다면 폴더에서 다음을 실행해 로컬 서버를 띄울 수 있습니다.

```cmd
cd /d d:\GameProject
python -m http.server 5500
```

그다음 브라우저에서 `http://localhost:5500` 접속.

또는 VS Code 확장 "Live Server"를 사용해 `index.html`을 Open with Live Server로 실행해도 됩니다.

## 구조 확장 팁
- 게임을 추가하려면 `js/games.js`에 새 클래스를 만들고, `main.js`의 `map`에 키를 등록하고, `index.html` 메뉴에 버튼을 하나 더 추가하세요.
- 에셋(이미지/사운드)을 쓰고 싶다면 `<img>`를 프리로드하고 `drawImage`를 사용하거나, 모듈을 쓰지 않는 방식으로만 로드하면 서버 없이도 동작합니다.

## 라이선스 / 주의
이 소스는 학습/프로토타이핑 용도의 예시 코드입니다. 원작 IP/리소스는 포함되어 있지 않으며, 상표/저작권에 유의하세요.
